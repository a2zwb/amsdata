# AMS - Academic Management System

A web-based Academic Management System for managing students, staff, attendance, and batches.

## Features

- **Student Management** - Add, view, and manage student records
- **Staff Management** - Manage staff members and their roles
- **Batch Management** - Organize students into batches with sub-batches
- **Attendance Tracking** - Mark and track daily attendance
- **Marks Management** - Record and manage student marks
- **Dashboard** - Overview of academic data and statistics

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Styling:** Tailwind CSS
- **Data:** JSON-based storage

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/a2zwb/amsdata.git
   ```

2. Open `index.html` in a web browser.

## Usage

1. Login with your credentials
2. Navigate through the dashboard to manage:
   - Students
   - Staff
   - Attendance
   - Batches
